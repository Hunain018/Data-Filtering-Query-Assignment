# Student Marks Sheet – Filtering & Queries (Pandas)
 
A Jupyter Notebook that answers 50 data filtering and query questions on a student marks dataset using **pandas**.
 
## Contents
 
- `Student_Marks_Filtering.ipynb` – notebook with all 50 questions, each as a markdown heading followed by a pandas code cell with output
- `std_marks_sheet.xlsx` – dataset (30 students, 7 subjects: Python, NumPy, Pandas, Streamlit, ML, DL, NLP, plus Total and Grade)
## Topics Covered
 
- Filtering rows by single/multiple conditions (`>`, `<`, `==`, `between`)
- Logical operators (`&`, `|`) for combined conditions
- Selecting specific columns with `.loc`
- Row-wise checks across multiple columns (`.any(axis=1)`, `.all(axis=1)`)
- Aggregate-based filtering (`.max()`, `.min()`, `.mean()`)
## How to Run
 
1. Open `Student_Marks_Filtering.ipynb` in Jupyter Notebook / JupyterLab / VS Code.
2. Make sure `std_marks_sheet.xlsx` is in the same folder.
3. Run all cells (`Run All`)
